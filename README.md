# Campus Microgreens & Nutrition Initiative (CMNI)

Campus Microgreens & Nutrition Initiative (CMNI) is India's first university-based nutrition, research, and innovation ecosystem. Initiated by **MicroBloom Enterprises** under founder **Kanchan N**, the pilot hub is established on campus at **Manipur University**.

CMNI integrates modular automated AgTech with student campus nutrition to improve academic focus parameters, drive local botanical research, and build practical vocational pathways.

---

## 🚀 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React Icons
- **PWA Features**: Service Worker caching, manifest.json, Add to Home Screen prompts.
